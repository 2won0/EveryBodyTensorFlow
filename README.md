## About
This repository provides tutorial python scripts used in the EverybodyTensorlfow lecture by Jaewook Kang.


### Compiler/Interface Dependencies
- Tensorflow >=1.4
- Python2 <= 2.7.12
- Python3 <= 3.6.0
